# Simulador de Planificación de Procesos

Proyecto que implementa los algoritmos FCFS y SPN con interfaz gráfica en Python.

## Características
- Lectura de procesos desde CSV
- Simulación en tiempo real
- Diagrama de Gantt dinámico
- Cálculo de métricas

## Tecnologías
- Python
- Tkinter
